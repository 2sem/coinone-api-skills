# Public REST V1 deprecated

Use only when a caller explicitly requests deprecated Coinone public endpoints.

Base URL: `https://api.coinone.co.kr`

## Endpoints

### orderbook-deprecated
- Path: `/orderbook`
- Method: `GET`

### ticker-deprecated
- Path: `/ticker`
- Method: `GET`

### ticker-utc-deprecated
- Path: `/ticker_utc`
- Method: `GET`

### recent-complete-orders-deprecated
- Path: `/trades`
- Method: `GET`

## Notes

- Prefer Public API V2 for all new work.
- Response naming differs from V2 and may use older field conventions.
